# kiro-prompt-library
**Zentrale Web-App für wiederverwendbare Kiro-Prompts**

## Features

✓ **Browse Prompts** - Durchsuchbare Prompt-Sammlung
✓ **Submit Prompts** - Community-Beiträge
✓ **Rating System** - Bewerte hilfreiche Prompts
✓ **Usage Tracking** - Sieh welche Prompts am meisten genutzt werden
✓ **Statistics** - Dashboard mit Metrics
✓ **Categories** - Organisiert nach Use Case
